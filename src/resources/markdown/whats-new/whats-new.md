### The long-awaited alpha version is here!

I know you guys have been waiting to get your hands on Pro Journey for a while now, and I am happy to be able to finally present the first public version.
This mod is a complete overhaul of LIGA: Esports Manager and may have features that the other game doesn't have - and the other way around.
That being said, as it is an alpha expect there to be bugs - if you find any feel free to report them in the discord.
- [discord.gg/ZaEwHfDD5N](https://discord.gg/ZaEwHfDD5N)

### Features

This version of LIGA: Pro Journey ships with a few core features that distinguish it from LIGA Esports Manager.
- Fully custom player & teams database (Thank you to everyone that helped with this!)
- New tournament schedule
- FACEIT pug system
- Many overhauled UI and UX systems
- Contract system for every player!
- Greatly improved bot mechanics

For more detailed information check out the github page or the discord channel.

**You are no longer a manager - it's all about you now.**
